**A Premium Interior Design & Furniture Portfolio Website.**

Mistri Interior Decor is a high-end portfolio website designed to showcase luxury interior architecture and bespoke furniture projects. Built with **Django** and styled with **Tailwind CSS**, it features an editorial-style layout, masonry image galleries, and a lead-generation system.

## üåü Features

* **Editorial "Zig-Zag" Layout:** Storytelling approach for showcasing key design concepts.
* **Masonry Gallery:** Pinterest-style vertical grid for project details.
* **Modern Luxury UI:** Custom Tailwind design with a "Muted Bronze" & "Stone" palette.
* **Parallax Hero Section:** Smooth scrolling visual effects.
* **Dynamic Filtering:** Filter projects by category (Interior, Furniture, Commercial) without page reloads.
* **Lead Generation:** Fully functional "Contact Us" form stored in the database with email notifications.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
